# Design Profissional

Este repositório serve como a base central para a organização e entrega da Avaliação A1 da disciplina de Design Profissional, focada em estabelecer uma presença digital e uma marca profissional coerente no mercado de TI.

### Descrição e Objetivos

Esta disciplina focou em entender a importância do posicionamento no mercado de trabalho, organização digital, e criação de evidências práticas para demonstrar engajamento com a área de tecnologia.

**Principais Objetivos Atingidos:**

* Demonstrar posicionamento profissional e pessoal.
* Desenvolver habilidades de organização digital e versionamento de código.
* Criar um Portfólio Digital como evidência prática do meu trabalho.

### Tecnologias e Ferramentas Utilizadas

* **Versionamento de Código:** GitHub
* **Web Development (Portfólio):** HTML5, CSS3
* **Ferramentas de Design:** Canva.
